EYELASHES

is a display typeface with unique features inspired from the shapes of eyelashes. It is a project for 24/FA DSD-3662-A course at SVA. It is originally designed to be used for window display signs but the usage is not limited to them. Initial sketch was done with a broad nib pen and following typeface drawn in Robofont tried to preserve the unique characteristics.

This website is dedicated to introducing the Eyelashes web typeface to everyone. Eyelashes are not exclusive to humans. Therefore, the Welcome page includes sound words for animals with eyelashes. It is suitable for book titles. More page includes more applications and sound words for library or theater settings. Lastly, on Preview page, the font can be viewed in two different scales with an option for dark mode.

by Kyubin Gil
